Objective
Reinforce HTML fundamentals by creating a structured, multi-page website that incorporates
document structure, navigation, and multimedia elements.

TASK
Create a properly structured HTML document using:
○ <!DOCTYPE html>, <html>, <head>, <body>
○ Include title and meta tags
● Add basic content:
○ Headings, paragraphs, and comments
● Develop multiple web pages:
○ Home, About, Contact pages and Link To External
● Implement navigation:
○ Internal links between the pages
○ External links to relevant websites
● Add multimedia:
○ Insert two images using the <img> tag, one from local and one from online(pick
one from Unsplash)
● Add contents that are organized into lists (either ordered or unordered list)
