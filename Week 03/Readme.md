Objective
Apply HTML structure and CSS styling using all three methods (inline, internal (document),
external), and multiple selector types

Requirements
1. HTML Structure
Create a single webpage with these sections:
● Header (name + title)
● Navigation menu (links to sections using anchors)
● About section (paragraph + image)
● Skills section (list)
● Daily Routine section (ordered list)
2. CSS Application
A. External CSS (main styling)
● Background color for page
● Typography (font-family, size)
B. Internal (Document Level) CSS (inside <style>)
● Style the navigation menu
● Style section headings differently
C. Inline CSS (have limited use for this)
● Apply to less than 5 elements
3. Selectors to use (must be demonstrated)
You must use:
● Element selector (e.g. p, h1 etc)
● Class selector (e.g. .section etc)
● ID selector (e.g. #header etc)
● Group selector (e.g. h1, h2, h3 etc)
● Descendant selector (e.g. nav ul li a)
● Pseudo-class (at least one: :hover)

CHALLENGES
- It was tricky to apply some methods of styling such as branch.
- For a webpager, we weren't aware that the Hashtags where used to link the section.
